## 一、本章诉求

上一节对用户的行为根据返利配置进行入账发送MQ消息，这一节将接收MQ消息开始结算返利。这里也就是给用户的活动账户充值。并提供一个日历签到返利的接口，用于后续对接到前端UI使用。
